# Sample files

Drop one or more test files here (for example: `.srt`, `.mp3`, `.wav`, `.mp4`).


## Suggested Windows flow

1. Double-click `..\scripts\start_all_servers.bat`.
2. Drag/drop one or more files in this folder onto `en_to_zh.bat` (or another preset script under `..\scripts\`).
3. Collect the generated subtitle output.
4. Double-click `..\scripts\stop_all_servers.bat` when you are done.
